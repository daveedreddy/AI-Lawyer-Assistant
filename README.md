# ⚖️ AI Lawyer Assistant

An AI-powered legal assistant built using Retrieval-Augmented Generation (RAG), ChromaDB, FastAPI, and React.

## 🚀 Features
- Legal document search
- Indian Acts & Constitution support
- Case law retrieval
- Semantic search using BGE-M3 embeddings
- ChromaDB vector database
- FastAPI backend
- React frontend

## 🛠️ Tech Stack

### Backend
- Python
- FastAPI
- ChromaDB
- Sentence Transformers (BGE-M3)
- LangChain

### Frontend
- React
- Vite
- Tailwind CSS
